## üìñ About The Project

CineNext is a sleek, modern web application designed for movie enthusiasts. It provides a clean and engaging interface to browse, search, and discover movies across a wide variety of genres by integrating the TMDB and OMDB APIs. From the latest Hollywood blockbusters to beloved K-Dramas, CineNext is your personal guide to the world of cinema.

Beyond discovery, the platform features an AI-powered recommendation system driven by the Gemini API, offering intelligent and personalized suggestions. With the addition of a real-time community chat system, CineNext has evolved into a social, full-stack ecosystem where users can connect and explore their passion for film together.

This project was built by the **PIXEL BITS** team as a showcase of modern web development skills.

<br />

## ‚ú® Features

* **üîçMovie Search:** A powerful search bar to find any movie by title.Integrated with TMDB and OMDB APIs to provide high-quality posters, accurate plot summaries, and up-to-date movie data from around the world.
  
* **ü§ñ AI-Powered Recommendation System:** Driven by the Gemini API, this feature analyzes user preferences to provide intelligent and personalized movie suggestions that go beyond basic keyword matching.
  
* **üí¨ Community Chat System:** A real-time social hub where users can discuss films, share reviews, and connect. Built with a robust PHP and SQL backend for persistent messaging. Also have emoji option.
  
* **üé≠Curated Categories:** Browse through hand-picked collections, including:
    * Recommended For You
    * Bollywood Hits
    * Regional Cinema
    * Hollywood Blockbusters
    * Action & Thrillers
    * Sci-Fi & Fantasy
    * K-Drama
* **üíªHero Section:** A stunning landing page with a featured movie to draw users in.
  
* **üì±Responsive Design:** Fully responsive layout that looks great on desktops, tablets, and mobile phones.
