## üìñ About The Project

**CineNext** is a sleek, modern web application designed for movie enthusiasts. It provides a clean and engaging interface to browse, search, and discover movies across a wide variety of genres. From the latest Hollywood blockbusters to beloved K-Dramas, CineNext is your personal guide to the world of cinema.

This project was built by the **PIXEL BITS** team as a showcase of modern web development skills.

<br />

## ‚ú® Features

* **Movie Search:** A powerful search bar to find any movie by title.
* **Curated Categories:** Browse through hand-picked collections, including:
    * Recommended For You
    * Bollywood Hits
    * Regional Cinema
    * Hollywood Blockbusters
    * Action & Thrillers
    * Sci-Fi & Fantasy
    * K-Drama
* **Hero Section:** A stunning landing page with a featured movie to draw users in.
* **Responsive Design:** Fully responsive layout that looks great on desktops, tablets, and mobile phones.


